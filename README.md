# üé¨ Movie Review Analysis using Simple RNN

This project performs **sentiment analysis** on movie reviews using a **Simple Recurrent Neural Network (RNN)** built with **TensorFlow and Keras**. The model classifies reviews as **positive** or **negative**, based on text input from the **IMDB dataset**.

---

## üìò Project Overview

Movie reviews often contain valuable insights into audience opinions. This project aims to automatically classify these reviews as positive or negative using **natural language processing (NLP)** techniques and a **Simple RNN model**.  
The RNN model captures the sequential nature of text, making it suitable for sentiment classification tasks.

---

## üß† Features

- Loads and preprocesses the **IMDB movie reviews dataset**  
- Tokenizes and pads sequences for uniform input length  
- Builds and trains a **SimpleRNN model** with embedding layers  
- Evaluates the model on test data  
- Predicts the sentiment of custom reviews
